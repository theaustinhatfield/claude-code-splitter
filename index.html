Paste this anytime to spawn your 2Ã—2 agent grid.

Copy & paste cat > /tmp/s.sh << 'EOF' tmux kill-session -t agents 2>/dev/null; sleep 0.5 tmux new-session -d -s agents tmux set -g mouse on tmux set -g pane-border-status top tmux set -g pane-border-format " #T " tmux send-keys 'claude' C-m; tmux select-pane -T "Agent-1" tmux split-window -h; tmux send-keys 'claude' C-m; tmux select-pane -T "Agent-2" tmux split-window -v; tmux send-keys 'claude' C-m; tmux select-pane -T "Agent-3" tmux select-pane -t 0; tmux split-window -v; tmux send-keys 'claude' C-m; tmux select-pane -T "Agent-4" tmux attach -t agents EOF bash /tmp/s.sh

Click any pane to select. Ctrl+B then D to detach.
